A clean, functional medicine search directory for the hospital pharmacy.

**DESIGN SYSTEM (REQUIRED):**
A clinical, trusting, and hyper-clean interface with confident asymmetric layouts and fluid spring-physics motion. The atmosphere is professional yet reassuring — like a modern, well-lit medical facility. Creativity is restrained, opting for clarity and data density but without feeling cramped.

- Canvas White (#F8FAFC) — Primary background surface
- Pure Surface (#FFFFFF) — Card and container fill
- Medical Charcoal (#0F172A) — Primary text, deep depth
- Muted Steel (#64748B) — Secondary text, metadata, descriptions
- Azure Trust (#0E7490) — Single accent for CTAs, active states, focus rings.
- Font: Plus Jakarta Sans for Display/Body, JetBrains Mono for codes/tables.
- Cards: Generously rounded (1.25rem). Diffused shadow. No 3-column equal grids.
- Anti-Patterns: No emojis, no Inter, no generic placeholders (use realistic medical names).

**Page Structure:**
1. Header: Back navigation, title "Tra cứu danh mục thuốc".
2. Search Bar: Large, prominent input field with search icon "Nhập tên thuốc hoặc mã định danh...".
3. Filter Tags: Quick filter pills (Thuốc kháng sinh, Thuốc tim mạch, Thuốc giảm đau).
4. Results Grid: A dense but clear list/data view mapping medicine items. Each item should display: Name (e.g. Paracetamol 500mg), Active Ingredient, Unit, and current Stock status.
5. Empty State / Pagination: Standard pagination layout.
